# SI^3

This is a Pytorch implementation of “Simple Yet Effective Selective Imputation for Incomplete Multi-view Clustering”

Run 
```
python main.py
```
or
```
python main.py --dataset 1 --missing_rate 0.5 --selection_ratio 0.5
```

## Requirements

- Python 3.11.13
- Pytorch 2.7.1
- scikit-learn 1.6.1
